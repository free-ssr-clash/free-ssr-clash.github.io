详细配置指南：一步步设置Shadowrocket

手动添加代理服务器

对于希望精细控制代理设置的用户，手动添加是最直接的方式：

  1. 打开Shadowrocket应用，点击底部导航栏的"配置"标签
  2. 选择"添加代理"选项

  3. 根据您的代理服务提供商提供的信息填写以下内容：

    • 类型：选择对应的协议（HTTP/HTTPS/SOCKS5/VMess等）
    • 地址：输入代理服务器的IP地址或域名
    • 端口：服务器监听的端口号
    • 用户验证（如需要）：填写用户名和密码
    • 加密方式（针对某些协议）：选择适当的加密算法

  4. 填写完成后点击"保存"，新代理将出现在您的代理列表中

导入配置文件

对于拥有复杂代理设置或订阅了多个节点的用户，导入配置文件是更高效的方式：

  1. 获取配置文件（通常为.conf或.json格式）：

    • 从您的代理服务提供商处下载
    • 或通过订阅链接自动获取

  2. 在Shadowrocket中导入配置：

    • 点击"配置"标签
    • 选择"扫描"（扫描二维码配置）或"导入"（从文件导入）
    • 按照提示完成导入过程

  3. 导入成功后，系统会自动解析配置文件并添加所有代理节点

节点选择与优化

配置完成后，选择最优节点对获得最佳体验至关重要：

  1. 节点测速：Shadowrocket内置测速功能，可以评估各节点的延迟和速度
  2. 地理位置考量：选择物理距离较近的节点通常能获得更好的速度
  3. 负载平衡：如果支持，可以设置自动切换节点以实现负载均衡
  4. 备用节点：配置备用节点以防主节点失效

高级安全设置：保护您的数字足迹

为了最大限度地保护您的隐私和安全，Shadowrocket提供了一系列高级设置选项：

  1. DNS设置：

    • 使用可信的DNS服务器（如1.1.1.1或8.8.8.8）

  2. 防泄漏保护：

    • 启用"防止WebRTC泄漏"选项
    • 关闭"本地中继"以减少潜在的安全风险

  3. 分流规则：

    • 设置智能路由，让特定应用或网站走代理，其他流量直连
    • 配置广告拦截规则提升浏览体验

  4. 全局与分应用代理：

    • 可选择全局代理（所有流量通过代理）
    • 或分应用代理（仅指定应用使用代理）

FAQ

SSR 的协议和混淆组合如何选择最适合的？
选择模拟 HTTPS 或 HTTP 的混淆方式，再搭配 AEAD 或协议插件，可以隐藏流量特征并提高抗封锁能力。不同网络环境需测试，选择最稳定和可靠的组合。
如何使用 Sing-box 配合 Clash Rules 实现精细分流？
将 Clash 的 rules 段提取出来转换为 Sing-box 的 routing 格式，或直接引用 rule-providers 源。Sing-box 的路由引擎支持更复杂的条件匹配，可结合 GeoIP 与 keyword 实现精细分流策略。
Netch 如何实现 UDP 游戏加速？
Netch 支持 UDP 转发，可专门为游戏流量建立高速通道。结合策略组和节点延迟检测，可有效减少丢包和延迟，提高游戏体验的流畅性和稳定性。
Trojan TLS 伪装域名如何提高抗封锁能力？
通过选择真实可访问域名并使用有效证书，Trojan 流量可以完全模拟 HTTPS 请求。隐藏真实流量特征，降低被封锁和 DPI 检测风险，保证节点长期可用。
Shadowrocket 节点延迟排序功能如何提升访问体验？
Shadowrocket 会测速节点延迟和丢包率，并优先使用低延迟节点。结合策略组和分流规则，可保证网页、视频和游戏流量通过最优节点，提高访问速度和稳定性。
WinXray 的 PAC 模式适合哪些场景？
PAC 模式通过规则动态决定流量是否走代理。适合访问国内外混合网站环境，无需手动切换节点，自动分流提高网络访问效率和连接稳定性。
QuantumultX 支持哪些定制化分流策略？
QuantumultX 支持域名、IP、端口、协议以及用户自定义脚本分流。结合策略组使用，可实现广告屏蔽、视频加速、游戏优化等功能，提高访问速度和个性化网络体验。
Hysteria2 多路复用在短连接中优势是什么？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适合高频短连接场景，如游戏和即时通讯，提升吞吐量和连接稳定性，优化网络体验。
V2RayN 的作用是什么？
V2RayN 是 V2Ray 的 Windows 图形客户端，提供节点管理、订阅导入、日志查看等功能。它支持多协议配置，并可与系统代理集成，是 Windows 用户最常用的代理管理工具之一。
Mihomo 节点分组如何自动切换最佳节点？
将节点按延迟、地区或用途分组，并启用策略组功能，系统会根据节点延迟和健康状态自动切换到最佳节点，保证网页、视频和游戏连接稳定流畅。
